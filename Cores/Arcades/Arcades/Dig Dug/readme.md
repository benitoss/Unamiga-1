Teclas:
F1: 1P START
F2: 2P START
F3: INSERT COIN

ESC: RESET

F4: Scanlines (alternando entre no scanlines, 25%, 50% y 75%)

Requiere el archivo DigDug.dat en la tarjeta SD del addon STM32.
